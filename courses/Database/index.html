Welcome to Database & SQL Course

Welcome to the Database Course.. In this course you will learn all about database, not just about theory but also about practical things. This course introduces concept and terminology in database like database architecture, relational model, Structured Query Language, database design, technologies and issues in database environment. It gives student basic knowledge about database system. It is prerequisite for Database Design course.

Learning outcomes:

  1. Describe database systems, terminology, environment, and new concept of database
  2. Apply database language and SQL Programming language
  3. Design database using structure data model
  4. Evaluate database processing and security mechanism
